# processos-pje
Programa que faz pesquisa no site do PJE pelo número da OAB e salva a informações de cada processo em uma página de uma planilha excel.

## Pré-requisitos

Antes de começar, certifique-se de ter instalado o seguinte:

- Python 3.11: [https://www.python.org/downloads/](https://www.python.org/downloads/)
- Git (opcional, mas útil para clonar o repositório): [https://git-scm.com/downloads](https://git-scm.com/downloads)
- Google Chrome: [https://www.google.com/intl/pt-BR/chrome/](https://www.google.com/intl/pt-BR/chrome/)

## Instalação

Siga os passos abaixo para configurar e executar o projeto:

1. Clone este repositório (ou faça o download do código-fonte):

    ```powershell
    git clone git@github.com:RafaFreitasDev/processos-pje.git
    ```

2. Navegue até o diretório do projeto:

    ```powershell
    cd processos-pje
    ```

3. Crie um ambiente virtual (recomendado):

    ```powershell
    python -m venv venv
    ```

4. Ative o ambiente virtual:

    - No Windows:

    ```powershell
    venv\Scripts\activate
    ```

    - No macOS e Linux:

    ```powershell
    source venv/bin/activate
    ```

5. Instale as dependências do projeto:

    ```powershell
    pip install -r requirements.txt
    ```

## Uso

Para executar o programa, use o seguinte comando:

```powershell
python main.py
```
